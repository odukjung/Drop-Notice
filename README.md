# LogDrop-Notice
Notice marker when item drop.
Notice to self => Whisper channle
Notice to Party => Notice channle

# Commands 
- **logdrop|ld** - turn off/on
- **logdrop|ld pt** - turn off/on Notice Party
- **logdrop|ld info** - check info
